# PGIM Deal Analysis Platform — Python dependencies
# Tested on Python 3.10+
#
# Install all at once:
#   pip install -r requirements.txt
#
# Runtime prerequisites (NOT pip packages — install separately):
#   Ollama  https://ollama.com            — local LLM inference (fallback only for mapping/classify)
#   Mapbox  https://account.mapbox.com   — static map rendering (token in deal config)
#
# Runtime DATA files (NOT pip packages — must be present, not installable):
#   backend/data/MasterPlan2025.geojson   — URA Master Plan land-use polygons.
#       Required for the SG Location-competitiveness score and for filling a comp's
#       Land Zoning from land use. Fully local/on-prem; a centroid cache
#       (_landuse_buckets.pkl) is built automatically on first use.
#   OneMap account (email + password)     — set in shared_settings.json for auto
#       token refresh; only needed for OneMap Themes (e.g. tourism proximity).
#       Public OneMap geocoding needs no token.
#
# Optional:
#   OpenAI API key — required for online comp search and GPT analysis models
#   Set via env var:  export OPENAI_API_KEY="sk-..."
#
# NOTE: The Location-competitiveness score and Land-Zoning-from-land-use features
# add NO new pip dependencies — they use only the standard library plus the
# GeoJSON data file above.
